# 📈 株取引Bot シミュレーター (ペーパートレード)

仮想資金 **$100,000** を使って、リアルの株価データで自動売買Botの成績を実験するためのローカルアプリ。
**リアルマネーは一切使用しません。**

## 構成

| 要素 | 技術 | 補足 |
|---|---|---|
| 株価データ | yfinance (Yahoo Finance) | 無料・APIキー不要。日足ベース |
| バックエンド | Flask (Python) | `http://127.0.0.1:5001` |
| 売買判断 | ルールベースBot | ゴールデンクロス + RSI。API費用ゼロ |
| データ保存 | SQLite3 (`simulator.db`) | 売買履歴・ポジション・資産推移 |
| UI | HTML + Chart.js | ダークテーマのダッシュボード |

## Botの売買ルール (simulator/strategy.py)

- **BUY**: SMA5がSMA20を上抜け(ゴールデンクロス)、または RSI(14) < 30(売られすぎ)
- **SELL**: SMA5がSMA20を下抜け(デッドクロス)、または RSI(14) > 70(買われすぎ)
- **資金管理**: 1銘柄あたり総資産の20%まで / 現金は総資産の2%以上を常に確保

## 戦略比較バックテスト

過去1〜10年の日足データで、**4戦略を同一条件で横並び検証**できる。全戦略が同じエンジン
(同じ資金管理・翌日始値約定・手数料0.1%)で走るため、シグナルの実力差だけが比較できる。

| 戦略 | 型 | シグナル |
|---|---|---|
| SMAクロス+RSI (現行/ライブBotと同じ) | トレンド+逆張り | SMA5/20クロス、RSI30/70 |
| モメンタム (12-1ヶ月) | 相対強度 | 月次で過去12-1ヶ月リターン上位半分を保有 |
| ブレイクアウト (Donchian 20/10) | トレンドフォロー | 20日高値ブレイクで買い、10日安値割れで決済 |
| バイ&ホールド | ベースライン | 全銘柄を買って持つだけ(月次で枠まで積増) |

- ライブBotより現実寄りの条件: シグナル判定は当日終値 → **約定は翌日始値**、**手数料0.1%/回**控除
- 指標: 総リターン / 勝率 / プロフィットファクター / 最大DD / シャープレシオ / 売買回数 / 手数料
- LLM判断戦略は対象外(過去日付の判断に未来の知識の混入を防げず、バックテストとして成立しないため)

また、資産推移グラフには **SPYベンチマーク線**(同じ資金で指数を買って持った場合)が表示され、「指数に勝てているか」が一目で分かる。

## 使い方

1. 「🤖 Botを実行」でウォッチリスト全銘柄のシグナルを判定し、条件を満たせば自動売買
2. 「自動実行」にチェックを入れるとブラウザを開いている間、10分ごとに自動実行
3. ウォッチリストは自由に追加/削除可能(米国株ティッカー。例: `AMD`, `PLTR`)
4. 売買履歴・判断理由・確定損益はすべてSQLiteに保存され、画面下部の表で確認可能
5. 「シミュレーションをリセット」で初期資金からやり直し

## 注意点 (仕組み上の制約)

- 日足ベースの判定なので、シグナルは1日1回程度しか変化しない。**数週間〜数ヶ月単位で回して成績を評価する**のが正しい使い方
- 手数料・スリッページ・税金は未考慮(実運用より有利な結果になる)
- yfinanceは非公式APIのため、短時間に大量リクエストすると制限がかかることがある(5分キャッシュで軽減済み)
